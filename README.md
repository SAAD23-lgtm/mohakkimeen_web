# Mohakkimeen Web Dashboard

لوحة تحكم تفاعلية عربية RTL لتحليل ردود استمارة المحكمين.

## الملفات

- `index.html` الصفحة الرئيسية.
- `style.css` التصميم الكامل + Responsive للموبايل.
- `script.js` التفاعل، الفلاتر، الشارتات، التصدير والطباعة.
- `data.js` بيانات الاستمارة بعد التنظيف.

## التشغيل المحلي

افتح ملف `index.html` مباشرة من المتصفح.

## الرفع على GitHub Pages

1. ارفع الملفات كما هي داخل Repository.
2. من Settings > Pages اختار Branch: `main` و Root.
3. افتح رابط GitHub Pages بعد النشر.

## تحسينات الموبايل في النسخة الحالية

- Top bar ثابت للموبايل.
- قائمة جانبية Off-canvas بزر فتح/إغلاق.
- تحويل جدول الردود إلى كروت سهلة القراءة على الشاشات الصغيرة.
- تحسين أحجام الشارتات والكروت والنصوص.
- دعم Safe Area للموبايلات الحديثة.
- تحسين قابلية اللمس للأزرار والفلاتر.
- تقليل الحركة تلقائيًا عند تفعيل إعدادات تقليل الحركة في الجهاز.
